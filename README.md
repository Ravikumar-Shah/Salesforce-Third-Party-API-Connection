# Salesforce-Third-Party-API-Connection Demo Project

A demo project: demonstrates how to connect Salesforce to 3rd party COVID-19 data provider.

## Concepts Covered

  - Component lifecycle  
  - Third party API connection  
  - Looping  
  - Searching/Filtering  
  - Tables  
  - SLDS inbuilt CSS  
  - Custom CSS  
  - Event handlers  
  - getters
